Prácticas y proyecto para la asignatura Animación

¿Qué puedo encontrar en este repositorio?
  - Rigging y modelado de personajes 3D
  - Animaciones con captura de movimiento
  - Animaciones con cinemática inversa y directa
  - Implementación de interpolaciones (lineal, Hermite, Catmull-Rom)
  - Automatizaciones para asignación de drivers
  - Implementaciones de control de rotación y de velocidad (cuaterniones)
  - Manipulación de fcurves
  - Manejo de keyframes manual y por código
  - Uso de fspy (implementado en blender con compositing y videoediting)
  - Renderizado de frames por capas (shadow catcher + fspy + 3D)

Tecnologías utilizadas:
  - Blender
  - Python
